================================================
		INSTALLATION
================================================

1. Place "ZvRadialBlendShape.pyc" in your script folder:

   Win XP: "C:\Documents and Settings\USERNAME\My Documents\maya\scripts"
   Win Vista/7: "C:\Users\USERNAME\Documents\maya\scripts"

2. Place the plugin "radialBlendShape.mll" in your plugin folder
   (create it if you don't have one):

   Win XP: "C:\Documents and Settings\USERNAME\My Documents\maya\plug-ins"
   Win Vista/7: "C:\Users\USERNAME\Documents\maya\plug-ins"

3. In Maya open the Script Editor and go to the Python tab, copy and paste these two rows below:

########################

import ZvRadialBlendShape
ZvRadialBlendShape.ZvRadialBlendShape()

########################

4. Highlight it and click on "Save Script to Shelf...", specify a name and click "OK".
